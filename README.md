# Quadratic-Equation-Solver
just enter the coefficients of the different degrees of the variables in the asked format, and get the solutions instantly of the equation.
For an equation of the form:
ax² + bx + c = 0

How It Works
The program uses the Quadratic Formula:
x = (-b +(-) √(b² − 4ac)) / 2a

It also considers if the equation has Real roots or Imaginary roots, Two equal roots or Two different roots.

Enter the values of a, b, and c when prompted

Example:
Input
a = 1
b = -3
c = 2

Output
Roots are: 2, 1
